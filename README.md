# VTOL-Strict-Landing-LuaScript
A custom ArduPilot Lua script for QuadPlane VTOL that enforces strict  two-phase precision landing — freezing altitude below 7m until the craft  is centered within 35cm of the target. No drift, no off-target landings.  Plug-and-play for Pixhawk. Drop into /APM/scripts/ and fly.
